# React-Router


## 什么是SPA?
* 一种web应用程序或网站
* 在和用户交互的时候当用户不会跳转到其他的页面
* 由JS实现URL变换和动态变换HTML的内容

## SPA的优点
* 速度快，第一次下载完成静态文件，跳转不需要再次下载
* 体验好，整体交互趋于无缝，更倾向于原生应用
* 为前后端分离提供了实践场所
